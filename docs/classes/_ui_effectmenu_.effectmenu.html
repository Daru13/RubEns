Class EffectMenu

UI element representing the effect menu.

It displays a list of effect buttons, whose effects are defined elsewhere, which can be applied on click.

    • Defined in UI/EffectMenu.ts:107

    Static method for creating an effect node, i.e. a node representing an effect in the UI, able to respond to a click by applying the related effect, openning a popup to let the user set the effect parameters if required.
